# MatrixClaw

Desktop AI agent platform powered by OpenClaw. Matrix-themed chat interface with multi-platform support.

## Features

- 🤖 Multiple AI models (OpenAI, Anthropic, Google, Local)
- 💬 Multi-platform messaging (Telegram, WhatsApp, Discord, etc.)
- 🔧 Skill system for extensibility
- 🔒 Privacy-first (local data storage)
- 🎨 Matrix-inspired UI

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build        # Build for current platform
npm run build:win    # Build for Windows
npm run build:mac    # Build for macOS
npm run build:linux  # Build for Linux
```
